# Sales Prediction Using Python

Project Overview
This project predicts product sales based on advertising expenditure on TV, Radio, and Newspaper using Linear Regression.

Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

Dataset
Advertising dataset containing advertising budgets and corresponding sales.

Model Used
- Linear Regression

Results
- High accuracy with strong correlation between TV advertising and sales.

Conclusion
TV advertising contributes the most to sales growth.
